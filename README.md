# SE302-Automation-Project

SE302 Course Project - Automation Exercise Testing

This project is an automated testing suite for the Automation Exercise website, developed as part of the SE302 - Software Testing and Maintenance course at IUS.

The project uses Playwright with JavaScript and follows the Page Object Model (POM) design pattern for maintainability and scalability.

 Project Structure

/pages - Contains Page Object classes (LoginPage, ProductsPage, etc.)
/tests - Contains the actual test scripts (project.spec.js)
playwright.config.js - Configuration settings for the test runner
